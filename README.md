 Project Task Management - Laravel + React + Inertia.js

## """"""""""""""""""""en-US"""""""""""""""""""""""""""""""""""
This project is a web-based task management application designed to support project managers in monitoring and managing tasks related to their projects.
Its purpose is to put into practice the knowledge acquired in PHP-Laravel and React.js with Inertia.js

##  Tech Stack
- **Backend** : Laravel 10
- **Frontend** : React.js with Inertia.js
- **Base de données** : SQLite
- **Authentification** : Breeze (Inertia) 

## Main Features
- Authentication (login)
- Dashboard: Overview of “to-do” and “in-progress” tasks by project
